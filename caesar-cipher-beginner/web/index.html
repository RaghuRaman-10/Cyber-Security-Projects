How it works

  1. Pick a shift k (the “key”). Each letter moves k steps forward (encrypt) or backward (decrypt) in A–Z or a–z.
  2. After Z you wrap to A. That wrap is modular arithmetic with modulus 26.
  3. Spaces, numbers, and punctuation are left alone so you can read real sentences.
  4. Crack: there are only 26 shifts for English letters, so trying every shift is instant — tiny key space.
