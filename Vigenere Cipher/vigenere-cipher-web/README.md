# Vigenère Cipher (HTML/CSS/JS)

This is a small cyber security / cryptography mini project that demonstrates the **Vigenère cipher** using a simple static web UI (no frameworks).

## Features

- Encrypt / decrypt using a keyword
- Preserves spaces, punctuation, and letter case
- Copy output, swap output → input, clear

## Run locally

Open `index.html` in your browser.

Tip: Press **Ctrl+Enter** (or **Cmd+Enter**) to run.

## Notes

The Vigenère cipher is a classical cipher and is **not secure** for modern communications. This project is for learning purposes.
